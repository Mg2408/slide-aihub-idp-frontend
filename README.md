# Frontend

This project now runs on Vite instead of Create React App.

## Scripts

- `npm run dev` or `npm start`: start the Vite dev server on port `3000`
- `npm run build`: create a production build in the `build` folder
- `npm run preview`: preview the production build locally

## Environment

Set the API base URL in `.env`:

```bash
VITE_AI_EXTRACT=http://localhost:8000
```
